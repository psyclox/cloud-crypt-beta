About Cl0ud Crypt

Cl0ud Crypt is a cutting-edge file encryption solution designed to protect your sensitive data in the cloud. Our Chrome extension provides robust AES-256 encryption before your files ever leave your device.

Developed as a final year project by a team of security enthusiasts, Cl0ud Crypt bridges the gap between convenience and security in cloud storage solutions. With zero-knowledge encryption, we ensure that only you can access your files - not even we can decrypt your data.

Technical Deep Dive

Our encryption process follows a rigorous security protocol:

  1. User selects files and provides password
  2. Password is processed through PBKDF2 with 100,000 iterations
  3. Random salt is generated for each encryption operation
  4. AES-256 GCM encryption is performed with the derived key
  5. Authentication tag and IV are stored with the encrypted file
  6. Encrypted file is uploaded to cloud storage

The system is designed with multiple layers of security:

  • Client-Side Only: All cryptographic operations occur in the browser
  • Zero Knowledge: No keys or unencrypted data leave the user's device
  • Memory Protection: Sensitive data is wiped from memory after use
  • Temporary Files: Encrypted temp files are securely deleted
